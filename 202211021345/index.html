环境信息

  • Centos 7 kernel 5.4
  • Mysql Server 5.7

Mysql Server 安装步骤

yum 方式安装

安装 MySQL 5.7 的 yum 源

如果因为源冲突，类似如下错误

可以卸载旧的 yum 源安装包，重新安装

检查 yum 源中的 mysql-community-server 信息

安装 mysql-community-server

启动 mysql server

配置 mysql server 开机启动，同时启动服务

MySQL5.7 会为 root 用户随机生成一个临时密码，密码记录在日志 /var/log/mysqld.log 中

使用临时密码登陆，修改密码，临时密码登陆后不能做任何操作，必须先更改密码
