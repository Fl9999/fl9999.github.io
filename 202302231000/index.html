tar 命令用法

发表于 2023-02-23 上层目录 Linux 阅读次数：

常用选项

选项 说明 示例
-c 打包文件
-x 解包
-t 检测打包文件中的内容 tar -tf test.tar
-f 目标文件名称，要打包或解包的文件名 tar -cf test.tar test
tar -xf test.tar
--exclude 打包时排除文件
打包的目录使用相对路径，排除的文件只能接相对路径
打包的目录使用绝对路径，排除的文件接相对路径或绝对路径		 tar -cf test.tar test --exclude=runtime/*

用法示例

排除指定文件或目录

  • 打包的目录使用相对路径，排除的文件只能接相对路径
  • 打包的目录使用绝对路径，排除的文件接相对路径或绝对路径

建议打包的目录和排除的文件使用的使用路径时，保持一致，都是用绝对路径，或都是用相对路径 [1]
