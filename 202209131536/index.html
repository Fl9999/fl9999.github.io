kubectl 常用选项

-n, --namespace='' 指定操作的 namespace
-A, --all-namespaces 列出所有的 namespace 中的资源
-o, --output= 输出格式，常用值包含：
- json
- yaml
- wide		 显示 Pod 详细信息
--show-labels 显示资源对象的标签 label
-l, --selector='' 使用标签选择算符选择对象 使用标签选择器列出 Pod
